# Theme selector demo content

Purpose: reusable demo content for the `html-slides` theme selector.

Use these as content specs when generating selector previews. Each file gives a
different deck idea and multiple slide types, so viewers can see what each theme
is good at instead of seeing six versions of the same layout.

## Theme demos

| Theme | Demo | Best-fit team | Main slide types |
|---|---|---|---|
| Micron dark executive | AI memory roadmap | Leadership / strategy | Narrative, strategic map, decision brief |
| Micron dark | Yield improvement review | Process / manufacturing engineering | Evidence, process, readiness gates |
| Micron light | Q3 capacity plan | Finance / ops / PMO | Summary, table, comparison, decision log |
| Guided learning | Microsoft Copilot for daily productivity | Training / enablement | Learn, recall, apply, practice |
| Brutalist | Operating principles refresh | Engineering / quality | Principles, signal log, action rules |
| Playful | New hire fab tour | Onboarding / site orientation | Tour map, quiz, checklist, activity |

## Build guidance

- Do not build every candidate slide. Pick 3-5 slides per selector preview.
- Make the preview show range: at least one visual/data slide, not only text.
- Keep claims mock/demo-safe. No real Micron confidential data.
- Selector previews should favor instantly recognizable theme behavior over full narrative depth.
