# Playful selector demo

Demo: New hire fab tour  
Team: Onboarding / site orientation  
Use when: people need energy, orientation, and a low-pressure way to participate.

## Slide candidates

```text
┌────────────────────────────────────────────────────────────┐

│                                                            │
│ Today: look, listen, ask, connect                          │
└────────────────────────────────────────────────────────────┘
```
