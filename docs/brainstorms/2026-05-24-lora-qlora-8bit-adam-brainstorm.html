Three memory levers, three different targets.

Proposal: explain these techniques through the memory buckets they touch, not as interchangeable fine-tuning tricks.

Why it matters: the team can adopt the right lever when memory blocks experimentation.

Credibility: mechanism-based explanation only. No benchmark, quality, or speed claims are made without source material.

04 - QLoRA Compresses The Base Under The Adapter

Extends LoRA by changing how the frozen base is stored.
