05 / In practice

A real, non-coding example

Let's say you have 80 photos from a team event and you need a simple album.

You say:

"Here's a folder of 80 photos. Rename them by date, pick the 12 best looking ones, and put those into a single PDF with two photos per page. Save it to my desktop as event-highlights.pdf."

What happens behind the scenes:

  • Opens the folder, reads file metadata, and renames each photo using the date taken.
  • Looks at every image, scores them for quality, and picks the top 12.
  • Writes a small script to lay them out two-per-page and exports a PDF.
  • Saves the file to your desktop and shows you a preview before finishing.

07 / Where to start

Try one tiny thing this week.

Pick a real chore

Something you'd actually like to stop doing manually.

Describe it like a person

Don't try to sound technical. Be specific instead.

Check the work

Open the file, scan the output, ask for changes if needed.

Build the muscle

Each task makes the next one easier — it's a habit, not a skill.

The best users aren't the most technical — they're the most curious.
