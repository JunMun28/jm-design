06 / rank

The rank knob controls adapter capacity

06
Low rank

Small file

Fast to train, easy to ship, but may underfit a complex style.

Balanced

Enough signal

Common starting point: capture the pattern before chasing bigger adapters.

High rank

More capacity

Can fit harder tasks, but raises memory and overfit risk.

Developer move

Start small, inspect outputs, increase rank only when the adapter clearly lacks capacity.

09 / dataset gate

09
1

2

3

4

10 / LoRA knobs

The knobs are simple; the interactions are not

10
Rank

Capacity of the adapter.

Alpha

How strongly the adapter update is scaled.

LR

How aggressively the adapter learns.

Practical rule

Change one variable at a time

For developers, the win is a repeatable experiment table: dataset version, config, sample prompts, observed failure, next change.

11 / where QLoRA enters

QLoRA is not the same shortcut as LoRA

11
LoRA

What gets trained?

Only adapter matrices. The base model stays frozen.

QLoRA

How is the frozen base stored?

The base LLM is loaded in low precision so fine-tuning fits smaller GPUs.

For brand illustration, teach LoRA first. For text-model fine-tuning on limited hardware, introduce QLoRA.
