Core concept

Two parts, always.

A recursive function has a base case, when to stop, and a recursive case, how to shrink the problem.

Base case

The smallest input the function can answer directly.

n == 0

Recursive case

Every other input delegates to a smaller version.

n * fact(n - 1)

Worked example

Factorial, seven lines.

def fact(n):
    if n == 0:
        return 1
    return n * fact(n - 1)

print(fact(4))  # 24

Trace fact(4). 4 * fact(3) -> 4 * 3 * fact(2) -> 4 * 3 * 2 * fact(1) -> 24.

Self check

You should get 3 / 3.

Check understanding

Which function recurses forever?

A

return 1 if n == 0 else n * f(n - 1)

Terminates: n shrinks toward zero.

B

return n + f(n + 1)

Correct. No base case is reached because n grows.

C

return n if n < 2 else f(n - 1) + f(n - 2)

Terminates for non-negative integers.
