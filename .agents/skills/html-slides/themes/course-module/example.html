Three forces fight on the wafer surface.

01

Joule heating

Current flowing through the wordline adds energy faster than the die can dissipate. Doubles roughly every 8°C.

02

Substrate conduction

Heat travels down through the bulk silicon to the package. The 1A stack uses a thinner substrate, which is faster, but it also concentrates flux.

03

Refresh tax

Hotter cells leak charge faster, so the controller refreshes more often, which heats the array more. A feedback loop with one stable point and one unstable point.

03 · Apply

Next time you see a thermal incident report, work through this list.

  • Capture the wordline current trace for the 30 minutes before the spike.
  • Check substrate thermistor delta: is the heat staying on the array or moving down?
  • Plot refresh interval against temperature. A reciprocal curve points to the feedback loop.
  • Compare against the matched 1Z control wafer; if 1Z stays cool, the issue is node-specific.
  • File the finding with the thermal envelope team within 24 hours.
