A specimen deck for the playful theme. Bright, friendly, ready to start the day.

For workshops, kickoffs, onboarding.

Use playful when the goal is to energise the room, not inform a board. The deck carries the warmth so the speaker can carry the substance.

Set expectations early: this theme is for energising a room. If the audience needs to be informed or persuaded with data, pick a quieter theme.

One accent per slide.

Rotate across the deck: coral, teal, sunshine, grape. Never all four on the same slide.

Point at the rotation rule — each slide picks one of the four accents. Using all four at once reads as confused, not playful.
