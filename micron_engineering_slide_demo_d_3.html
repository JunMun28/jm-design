14 / Decision log
Lessons learned

What we learned from the pilot

What worked

  • Users understood the planning workflow quickly
  • Role separation between admin and assignee was clear
  • Real-time assignment visibility reduced follow-up questions

What to improve

  • Warning messages need clearer wording
  • Exception ownership should be more visible
  • Training should include one full end-to-end example

Next iteration

Add clearer exception labels, shorten training content, and include screenshots for the top three user actions.

Closing
