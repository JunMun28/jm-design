07 - What are agent skills?

Agent Skills.

Reusable workflows that teach AI agents how to complete tasks properly.

Inside a Skill
01
Purpose
What task to complete
02
Steps
How the work should happen
03
Rules
What to follow or avoid
04
Tools
What systems it can use
05
Output
What good result looks like

Skills make agent behavior consistent, repeatable, and reliable.

08 - Progressive discovery

The agent discovers the right skill as it works.

Example: Turn this folder of notes into slides.

1 User goal

"Turn this folder of notes into slides."

The goal is clear. The method is not.

2 Scan skill descriptions

  • Slide Drafting - create slide outlines
  • Meeting Summary - action items
  • Data Analysis - explain trends
  • Email Writing - draft messages
  • Browser Task - complete web workflow

Scan first. Open what matters.

3 Open selected skill

Slide Drafting

  • Steps
  • Rules
  • Tools
  • Examples
  • Output format

4 Apply + output

Slide draft ready.

The agent follows the selected skill to produce the result.

Progressive discovery keeps the agent efficient: scan first, open what matters, then act.
